Atalhos no VSCode

Ctrl+O = Abrir documeto existete.

Ctrl+K+Ctrl+O (Ctrl+K+O) = Abrir arquivos.

Ctrl+B = Minimizar explorador.

Ctrl+F = Pequisar dentro do documento (tags, class, textos etc.)

Ctrl+H = Mudar qualquer nome dentro de um documento. Se você utilizar o Ctrl+Alt+Enter após selecionar o nome ser mudado, mudará todos os nomes de uma vez.

Ctrl+; = comentar (linha vazia, linha cheia e tags completas), para desfazer o comentário basta refazer o mesmo talho.

Shift+alt+baixo ou cima = duplicar linha.

alt+baixo ou cima = mover linha.

Ctrl+D = Alterar tags, palavras ou textos iguais.

Ctrl+L = selecionar linha.

Alt+Z = Retirar ou colocar scroll (Quebra de linhas)

Ctrl+TAB = Navegar entre abas do VSCode.

Ctrl+Baixo ou Cima = Mover Scrollbar.

Ctrl+G = Caçar linha específica.
